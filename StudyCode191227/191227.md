# DOM Ready

- ('')は文字列

# HTTPS対応にする理由
- 送信するときにHTTPのままだとそのまま文字列として送信されるのでHTTPS対応は必須
- HTTPSは送信時、自動で暗号化してくれる
- GoogleはHTTP対応のサイトはアップしてくれない
- 「常時SSL設定」が必要
